# 商品筛选测价环节快速定价联动脚本

## `ui-screenshots.mjs`

用途：启动临时工作台服务，使用临时数据库复制件跑通商品加入筛选池、进入测价、测价通过三段链路，并生成 UI 截图留档。

运行：

```bash
node scripts/商品筛选测价环节快速定价联动/ui-screenshots.mjs
```

输出：

- `docs/测试文档/商品筛选测价环节快速定价联动/UI/selection-quick-pricing-default.png`
- `docs/测试文档/商品筛选测价环节快速定价联动/UI/selection-quick-pricing-pricing.png`
- `docs/测试文档/商品筛选测价环节快速定价联动/UI/selection-quick-pricing-result.png`
