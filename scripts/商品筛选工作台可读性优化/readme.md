用途：启动本地工作台服务，使用临时 SQLite 副本生成商品筛选工作台卡片列表截图。

```bash
node scripts/商品筛选工作台可读性优化/ui-screenshots.mjs
```

- `docs/测试文档/商品筛选工作台可读性优化/UI/selection-card-list.png`
