# 萌拉结果工作台与商品筛选页优化脚本

## 目录说明

- `ui-screenshots.mjs`：启动临时工作台服务，基于 Playwright 生成本地 UI 回归截图

## 使用方式

在仓库根目录执行：

```powershell
node scripts/萌拉结果工作台与商品筛选页优化/ui-screenshots.mjs
```

## 产物目录

截图会输出到：

```text
docs/测试文档/萌拉结果工作台与商品筛选页优化/UI/
```

当前默认生成：

- `results-default.png`
- `selection-processing.png`
- `selection-ready.png`
