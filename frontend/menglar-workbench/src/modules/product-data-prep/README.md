# Product Data Prep Module

这个目录是“商品数据整理”模块的独立前端工作区。

## 目录说明
- `ProductDataPrepWorkbench.jsx`：模块入口，页面只做挂载。
- `components/`：模块内 UI 组件。
- `data/`：计划、字段分组、边界说明。
- `mock/`：前期联调用的候选商品与草稿样例。
- `api/`：模块内 API 封装，后续接真实后端时优先在这里扩展。

## 当前阶段
- 保持独立路由，不回写结果页。
- 先用模块内 mock 和后端占位接口稳定字段契约。
- 下一步优先补候选商品、草稿编辑、校验和导出链路。
