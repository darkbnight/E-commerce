# 早期跨境选品过滤规则

本规则用于在“机会指数初筛”之后，对候选子类目做人工逻辑过滤。它不是打分模型，也不做复杂扣分；目标是快速剔除不适合早期跨境电商验证的类目。

## 一、使用位置

完整流程：

```text
萌啦数据采集 -> 机会指数初筛 -> 早期跨境过滤规则 -> 初步结论 -> 三级类目下钻 -> 商品分析流程交接
```

机会指数回答：

```text
这个子类目从数据上看有没有需求和市场空间？
```

早期跨境过滤规则回答：

```text
这个子类目适不适合早期团队低成本验证？
```

两者不能混为一谈。数据好但业务风险高的类目，应放入“谨慎观察”或“暂不优先”。

## 二、核心过滤规则

### 1. 物流友好

优先选择：

- 小件
- 轻量
- 非液体
- 非粉末
- 不易碎
- 单位体积价值较高
- 包装标准化，运输损耗低

尽量避免：

- 大件、重货
- 液体、膏体、粉末
- 易碎品
- 强气味商品
- 带电池或复杂电器
- 对仓储温湿度敏感的商品

判断原则：物流成本和运输不确定性会直接吞掉早期利润，不适合一开始就挑战复杂品。

### 2. 产品简单

优先选择：

- 用途清楚
- 参数少
- 规格关系简单
- 不需要大量专业知识
- 用户看图就能理解
- 标题、图片、详情页能讲清楚卖点

尽量避免：

- 专业性强的工具、耗材、配方类商品
- 参数多、适配关系复杂的商品
- 看不懂用途或使用条件的商品
- 对说明书、安装、调试依赖高的商品

判断原则：看不懂、讲不清、容易用错的商品，不适合早期验证。

### 3. 退货和纠纷低风险

优先选择：

- 不涉及强安全责任
- 不依赖明显效果承诺
- 不需要复杂安装
- 不容易因尺寸、适配、材质产生纠纷
- 售后解释成本低

尽量避免：

- 安全要求高的商品
- 易产生“没效果”争议的商品
- 强依赖尺寸适配的商品
- 易损、易坏、易误用的商品
- 售后判断成本高的商品

判断原则：早期不要碰容易维权、退货、差评和解释成本高的类目。

### 4. 合规低风险

优先选择：

- 普通消费品
- 无特殊认证要求
- 非食品、非药品、非化学品
- 非儿童安全强监管品
- 非医疗、保健、功效承诺类商品

尽量避免：

- 化学品
- 肥料、农药、土壤改良剂
- 种子、幼苗、植物相关敏感品
- 食品接触高风险商品
- 儿童安全强监管商品
- 带电、发热、切割、压力类商品

判断原则：合规不确定性高的类目，即使数据漂亮，也不适合早期验证。

### 5. 国内供应链有优势

优先选择：

- 国内供应成熟
- 成本有优势
- 质量可控
- 可稳定补货
- 可做套装、规格、包装或设计差异化

尽量避免：

- 本土品牌强势的类目
- 国内无明显成本或产品优势的类目
- 质量标准难控的类目
- 只能靠低价竞争的同质化类目

判断原则：跨境不只看需求，还要看国内供应链是否能打出优势。

### 6. 差异化可表达

优先选择：

- 图片能表达优势
- 标题能表达核心卖点
- 规格、套装、材质、场景有差异化空间
- 用户能快速理解为什么买它

尽量避免：

- 同质化严重
- 只能拼价格
- 卖点需要大量教育
- 优势无法通过图片和文案表达

判断原则：早期选品要能被页面表达出来，否则转化和广告效率都会受影响。

## 三、典型剔除样例

### 土壤、化肥和园艺化学品

即使产品数低、销量和销售额不弱，也不适合作为早期优先验证类目。

主要原因：

- 可能涉及粉末、液体、化学成分或植物营养剂
- 物流、清关、平台审核不确定性高
- 使用方法、剂量、适用场景有专业门槛
- 容易出现“无效果”“烧苗”“使用不当”等售后争议
- 合规和标签要求可能复杂

结论：放入“暂不优先”或“高风险观察”，不放入第一批优先验证。

### 种子和幼苗

数据上可能出现高销量、低产品数，但早期仍需谨慎。

主要原因：

- 植物和种子类可能有检疫、清关、平台规则风险
- 季节性强
- 损耗和发芽效果难控
- 客单价可能偏低

结论：数据可继续观察，但不适合无经验团队早期直接验证。

## 四、快速检查清单

进入三级类目下钻前，先问：

- 是否小件、轻量、非液体、非粉末、不易碎？
- 是否无需专业知识就能理解和使用？
- 是否不涉及明显安全责任、功效承诺或复杂售后？
- 是否没有明显合规、认证、清关、平台禁限售风险？
- 国内供应链是否有成本、质量、款式或响应速度优势？
- 商品优势是否能通过图片、标题、规格和场景表达出来？

如果多个问题答案是否定，即使机会指数高，也不要放入第一批验证。
