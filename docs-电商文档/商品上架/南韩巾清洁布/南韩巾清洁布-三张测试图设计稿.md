## 3. 图片设计

### 图1：主图
- 对应素材：`1.jpg`
- 目标：
  - 主图
- 建议处理：
  - 保留右侧灰色布主体
  - 左侧带韩文包装弱化、裁切或避免成为视觉中心
  - 背景保持干净，不堆叠过多图标和气泡
- 建议文案：
  ```text
  30x40 см
  2 шт
  Для стекол и зеркал
  Без разводов
  ```
- 文案中文释义：
  - `30x40 см`：30x40 厘米
  - `2 шт`：2 片装
  - `Для стекол и зеркал`：适用于玻璃和镜子
  - `Без разводов`：不易留水痕
- 版式建议：
  - 产品主体居中或偏右
  - 左上：`30x40 см`
  - 左下：`2 шт`
  - 右上或右下：`Для стекол и зеркал`
  - 小字补充：`Без разводов`
- 备注：
  - 不写 `корейская`
  - 不写 `микрофибра`
  - 不写过长卖点文案

### 图2：擦拭表现图
- 对应素材：`2.jpg`
- 目标：
  - 擦拭表现图
- 建议处理：
  - 去掉原中文文案
  - 保留擦玻璃动作画面
  - 用俄语重新覆盖文字
- 建议文案：
  ```text
  Не оставляет заметных разводов
  Удобно для ежедневной уборки
  ```
- 文案中文释义：
  - `Не оставляет заметных разводов`：不易留下明显水痕
  - `Удобно для ежедневной уборки`：适合日常清洁使用
- 版式建议：
  - 保留右上文字区
  - 标题大字：`Не оставляет заметных разводов`
  - 次级说明一行：`Удобно для ежедневной уборки`
- 备注：
  - 本图不再重复 `30x40 см`
  - 本图不再重复 `2 шт`
  - 本图不再重复“适用于玻璃和镜子”

### 图3：吸水/质感图
- 对应素材：`3.jpg`
- 目标：
  - 吸水/质感图
- 建议处理：
  - 保留布面弯曲和接触水面的画面
  - 文案控制在 1-2 句内
- 建议文案：
  ```text
  Впитывает влагу
  Мягкая текстура
  ```
- 文案中文释义：
  - `Впитывает влагу`：吸水
  - `Мягкая текстура`：质地柔软
- 版式建议：
  - 文字放左上或右上
  - 避免覆盖布面细节
  - 这一张只讲物理特征，不再讲太多用途
## 4. 增强测试版补充图设计

### 图4：多表面适用图
- 参考方向：原补充图中的“通用适配多种表面”结构
- 目标：
  - 多表面适用图
- 建议文案：
  ```text
  Подходит для разных поверхностей
  Металл
  Плитка
  Дерево
  Стекло
  Пластик
  ```
- 文案中文释义：
  - `Подходит для разных поверхностей`：适用于多种表面
  - `Металл`：金属
  - `Плитка`：瓷砖
  - `Дерево`：木质表面
  - `Стекло`：玻璃
  - `Пластик`：塑料
- 版式建议：
  - 顶部大标题：`Подходит для разных поверхностей`
  - 下方使用 4-5 个小场景宫格
  - 场景图优先展示与商品强相关的表面
- 备注：
  - 不需要把所有场景写满
  - 玻璃、瓷砖、木质、金属、塑料优先

### 图5：吸水强化图
- 参考方向：原补充图中的“吸水表现图”结构
- 目标：
  - 吸水强化图
- 建议文案：
  ```text
  Впитывает влагу
  Подходит для ежедневной уборки
  ```
- 文案中文释义：
  - `Впитывает влагу`：吸水
  - `Подходит для ежедневной уборки`：适合日常清洁
- 版式建议：
  - 顶部大标题：`Впитывает влагу`
  - 中间使用布体与水接触的直观画面
  - 下方只保留 1 行辅助文案，不再加夸张数据
