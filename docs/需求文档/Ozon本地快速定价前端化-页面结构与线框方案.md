# Ozon本地快速定价前端化-页面结构与线框方案

## 页面入口
- 左侧导航新增 `Ozon 快速定价`。
- 路由为 `/ozon-pricing`。

## 页面结构
```text
┌──────────────────────────────────────────────────────────────┐
│ 顶部说明区：Ozon Local Pricing / 快速定价说明 / 汇率快照       │
├──────────────────────────────┬───────────────────────────────┤
│ 左侧参数区                    │ 右侧结果区                    │
│ - 类目佣金选择                │ - 折前价、折后价               │
│ - 采购成本、重量              │ - 利润、利润率、计费重量       │
│ - 毛利类型和值                │ - 收入明细                     │
│ - 前台折扣                    │ - 成本明细                     │
│ - 物流方式、交货类型          │ - 风险提示                     │
│ - 境内段运费、广告费率等      │                               │
│ - 体积参数                    │                               │
│ - 手动物流费/佣金率覆盖       │                               │
│ - 重置、CEL 物流对比按钮      │                               │
└──────────────────────────────┴───────────────────────────────┘

弹窗：CEL 物流费用对比
┌─────────────────────────────────────┐
│ 标题 / 关闭按钮                      │
├───────────────┬─────────────────────┤
│ 物流商         │ 取货点 / 上门费用    │
└───────────────┴─────────────────────┘
```

## 主要交互
- 表单字段变更后即时重新计算，不强制用户点击计算按钮。
- `重置` 恢复原工具默认参数。
- `查看 CEL 物流费用对比` 打开弹窗，使用当前重量、体积参数分别计算取货点和上门配送费用。
- 手动物流费不为空时覆盖自动物流报价；手动佣金率不为空时覆盖类目佣金率。
- 当费率或毛利目标导致不可定价时，结果区显示风险提示，避免静默输出异常数值。

## 响应式
- 桌面端为左右两列工作台。
- 窄屏下参数区和结果区上下堆叠，弹窗内容单列展示。
