# 商品视频生成工具

## 背景

用户已有「图片压缩」工具，压缩后的 JPG 图片散落在各商品目录中。需要一个简单工具，输入目录列表，把每个目录下的图片拼成幻灯片 MP4 视频。

## 使用方式

1. 在「图片目录」textarea 中每行输入一个目录路径
2. 设置视频时长（默认 12 秒，范围 5-300 秒）
3. 选择输出分辨率（1080p / 720p）
4. 自定义输出视频名（默认「商品视频」）
5. 点击「开始生成」

每个目录独立生成一个 MP4 视频，输出到该目录下。

## 处理逻辑

- 读取目录下所有支持的图片格式（JPG / JPEG / PNG / WEBP / BMP）
- 按文件名自然排序
- 每张图均分视频时长（如 5 张图 × 12 秒 = 每张 2.4 秒）
- 所有图片统一缩放到目标分辨率，保持原始宽高比，黑边填充
- 非 JPG 格式先用 ffmpeg 转为 JPG，再合成视频
- 使用 libx264 编码，yuv420p 像素格式，30fps 输出

## 页面结构

参照「图片压缩」页面：

- 左侧 Panel：视频参数表单（目录输入、时长、分辨率、视频名）
- 右侧 Panel：生成结果（视频数量、图片总数、每目录详情）

## 技术要点

- 与图片压缩工具共用 ffmpeg 环境，通过 `FFMPEG_PATH` 环境变量配置
- 处理过程中使用临时目录存放中间帧，完成后自动清理
- 支持 CLI 独立运行：`node generate-product-video.mjs --dirs=dirA,dirB --duration=15`
