# 商品内容资产页面 - 页面结构与线框方案

## 页面定位
本页面定位为商品内容资产浏览器，不再默认展示某一个商品的完整详情。主页面负责浏览、筛选和定位；商品详情进入右侧抽屉。

## 首屏结构
```text
左侧导航

┌────────────────────────────────────────────────────────────┐
│ 商品内容资产浏览器                                          │
│ 按商品浏览标题、描述、标签、图片、SKU 图片和历史版本          │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 筛选：平台 | 商品ID/标题 | 内容状态 | 历史版本 | 重置          │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 商品数 | 待处理 | 缺图片 | 缺SKU | 今日新增 | 最近采集        │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 主图 | 商品标题 | 平台 | 商品ID | 状态 | 图片 | SKU | 标签 | 版本 │
│ 行 1                                                       │
│ 行 2                                                       │
│ 行 3                                                       │
└────────────────────────────────────────────────────────────┘
```

## 详情抽屉结构
点击表格行后，从右侧打开详情抽屉。

```text
┌──────────────────────────────┐
│ 状态  商品标题                │
│ 平台 / 平台商品ID             │
├──────────────────────────────┤
│ 概览 | 内容 | 图片 | SKU | 版本 | 经营 │
├──────────────────────────────┤
│ 当前 Tab 内容                 │
└──────────────────────────────┘
```

## Tab 内容
- 概览：图片数、SKU 数、标签数、版本数、完整性检查项、版本 ID、采集时间、内容哈希。
- 内容：标题、描述、标签。
- 图片：主图、图库。
- SKU：SKU 名称、平台 SKU ID、价格、SKU 图片。
- 版本：历史版本列表，点击后切换当前详情版本。
- 经营：销量、销售额、均价、毛利率、店铺、品牌、曝光、点击、转化率、物流。

## 交互原则
- 主页面只展示判断和定位字段，不展示长描述和完整图片墙。
- 详情抽屉保留列表上下文，适合连续核查多个商品。
- 内容状态前置，用户优先看到哪些商品需要处理。
- 版本是商品的子信息，只在详情抽屉内展示。
