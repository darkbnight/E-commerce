# 商品数据整理工作台测试用例

## 1. 测试范围
本轮覆盖“商品数据整理”侧边栏页签、`/product-data-prep` 新路由、页面默认态和开发边界说明。

本轮不覆盖真实候选商品导入、草稿保存、数据库写入、图片处理、属性映射和 Ozon 上货工具联动。
