## 页面主链路测试

- [x] 打开商品结果页，选择有商品数据的批次，表格展示商品图、标题、商品 ID、创建日期。
- [x] 表格展示品牌、店铺名称、商品类型。
- [x] 表格展示平均单价卢布/人民币、销售金额卢布/人民币、点击率、广告费和广告占比。
- [ ] 关键词输入店铺名或商品标题后，列表能返回匹配商品。

## API 接口测试

- [x] 请求 `GET /api/products?pageSize=5` 返回 200。
- [x] `items[0]` 包含 `product_image_url`、`shop_name`、`avg_price_rub`、`avg_price_cny`、`ad_cost_cny`、`product_created_date`。
- [x] `items[0]` 不包含旧字段 `ozon_product_id`、`sales`、`revenue`。

## 数据库存储与写入测试

- [x] 表 `product_business_snapshots` 存在字段 `product_image_url`，预期值来自 `products_raw.raw_payload.skuImg`。
- [x] 表 `product_business_snapshots` 存在字段 `shop_name`，预期值来自 `products_raw.raw_payload.shopName`。
- [x] 表 `product_business_snapshots` 存在字段 `avg_price_rub`，预期值来自 `products_raw.raw_payload.avgPrice`。
- [x] 表 `product_business_snapshots` 存在字段 `avg_price_cny`，预期值来自 `products_raw.raw_payload.avgPriceRmb`。
- [x] 表 `product_business_snapshots` 存在字段 `product_created_date`，预期值来自 `products_raw.raw_payload.createDt`。
- [x] 关联关系：`product_business_snapshots.raw_record_id = products_raw.id`。

## 异常与边界测试

- [x] 原始响应缺少图片时，页面展示占位图，不影响表格布局。
- [ ] 原始响应缺少人民币字段时，页面对应字段显示 `-`。
- [x] 老数据库缺少新增列时，启动 API 或执行迁移脚本后自动补列。
