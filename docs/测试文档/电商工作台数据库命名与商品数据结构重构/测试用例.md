# 电商工作台数据库命名与商品数据结构重构测试用例

## 1. 测试范围

本轮覆盖以下变更：

- 数据库默认路径切换为 `db/ecommerce-workbench.sqlite`。
- 新增 `product_business_snapshots` 商品经营快照表。
- 新增 `product_content_assets` 商品内容资产表。
- 将旧 `products_normalized` 数据迁移到 `product_business_snapshots`，迁移完成后删除旧表。
- 后端结果接口从 `product_business_snapshots` 查询，同时保持前端返回字段兼容。
- 结果展示页继续可展示商品，并可将当前页加入商品筛选池。
